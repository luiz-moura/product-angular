## Run Locally

Clone the project

```bash
  git clone https://github.com/luiz-moura/product-angular.git
```

Go to the project directory

```bash
  cd product-angular
```

Install dependencies

```bash
  npm install
```

Start the back-end server

```bash
  npm run server
```

Start the project

```bash
  npm start
```
